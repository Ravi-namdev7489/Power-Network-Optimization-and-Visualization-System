# Power Network Optimization & Visualization System

**Technologies:** Python (Flask, PyPSA, Pandas, NumPy, Matplotlib), Folium, MySQL, Bootstrap, HTML, JS  
**Tools:** VS Code, Jupyter Notebook, GitHub  

## Description
A full-stack web application for **power-network modeling, optimization, and visualization**.  

**Key Features:**
- Developed a **Flask-based web application** integrating PyPSA for power network optimization.
- Implemented **CSV upload system** for network data including buses, lines, loads, and transformers.
- Performed **network optimization** and analyzed energy flow dynamically.
- Built an **interactive Folium map** showing substations, line connections, and network visualization.
- Added **secure user authentication** using Flask-Session with MySQL backend.
- Generated **bar and area plots** to analyze generator and storage performance.
